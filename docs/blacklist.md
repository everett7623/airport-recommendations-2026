## 🔍 用户如何自查机场风险？

在购买前建议检查：

### 1）是否支持月付
- 月付是风险最低的选择
- 避免一次性大额年付

### 2）是否有公开社区讨论
- V2EX / Reddit / Telegram 群组
- 是否存在大量负面投诉

### 3）运营是否透明
- 是否长期更新公告
- 是否有明确工单/客服渠道

### 4）是否提供试用
- 支持试用的机场一般更可信

## ✅ 最佳风险控制策略（强烈建议）

- 主力机场优先选择月付专线或当前 README 中的性价比/高端专线推荐
- 备用机场优先选择按量计费、不限时流量或无 AFF 备选
- 永远不要只依赖单一机场
