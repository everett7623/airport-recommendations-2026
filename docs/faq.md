# ❓ 机场常见问题解答 (FAQ) - 新手必读指南

> **前言：** 本文档汇集了用户在使用机场过程中 99% 会遇到的问题。  
> 如果您是第一次接触科学上网，建议通读全文，可以帮您少走大量弯路。

## 🔥 热门应用篇（ChatGPT / TikTok / Netflix）

### Q1：为什么连上了梯子，ChatGPT 还是提示 Access Denied？

**A：ChatGPT 对 IP 风控非常严格，常见原因如下：**

1. **节点不支持解锁**  
   请优先选择带有 `[ChatGPT]`、`[OpenAI]`、`[AI]`、`[解锁]` 标记的专用节点。

2. **IP 不干净（共享用户太多）**  
   热门美国节点容易被滥用，建议切换：
   - 日本 / 新加坡 / 台湾  
   - 冷门地区（加拿大、澳洲）

3. **浏览器缓存泄露定位信息**  
   建议：
   - 使用无痕模式访问  
   - 清理 Cookie

4. **模式问题**  
   新手建议临时切换到：
   - **Global 全局模式**  
   测试成功后再切回 Rule 规则模式。

---

### Q2：为什么 TikTok 黑屏/刷不出内容？

**A：TikTok 是目前风控最严格的平台之一（SIM + IP 双检测）。**

✅ 必做步骤：

1. **拔卡（最稳）**  
   TikTok 会识别 +86 SIM 卡，建议拔卡或使用国外废卡。

2. **必须原生 IP 节点**  
   推荐使用支持原生 IP 的机场（光速云、TNT、星岛梦等）。

3. **地区一致**  
   美区 ID 下载 TikTok → 建议配美国节点  
   日区 ID → 配日本节点

---

### Q3：Netflix 只能看自制剧？怎么解锁全部片库？

**A：这是“广播 IP”导致的。**

- **原生 IP（Native IP）**：Netflix 认为是家庭宽带 → 全片库  
- **广播 IP（Broadcast IP）**：Netflix 认为是代理 → 只剩 Originals  

✅ 想解锁完整片库，请认准“原生 IP 专线”。

## 🧐 基础概念篇（小白扫盲）

### Q4：机场和 VPN 有什么区别？

| 项目 | 机场（SS/V2Ray/Trojan） | VPN（传统 OpenVPN） |
|------|-------------------------|----------------------|
| 速度 | 更快，延迟更低 | 相对更慢 |
| 解锁流媒体 | 更强 | 一般 |
| 抗封锁 | 更灵活 | 容易被封 |
| 使用难度 | 稍复杂 | 更傻瓜 |

✅ 国内主流推荐使用机场。

---

### Q5：什么是“倍率”？为什么流量扣得很快？

倍率 = 流量扣除倍数：

- **1x**：用 1GB 扣 1GB（常规节点）
- **0.5x**：用 1GB 扣 0.5GB（优惠节点）
- **3x**：用 1GB 扣 3GB（极速专线/IPLC）

💡 建议：
- 日常用 1x
- 下载选低倍率
- 游戏/专线可接受高倍率

---

### Q6：IPLC / IEPL 专线是什么？为什么贵？

- 普通线路：走公网 → 晚高峰拥堵、易丢包
- IPLC/IEPL：物理专线 → 不过墙、不受 GFW 干扰

✅ 优势：
- 敏感时期依然稳定
- 延迟极低（游戏首选）
- 晚高峰 4K 不掉速

---

### Q7：什么是“节点倍率 0.8-1”？怎么看？

机场常写：

- `香港 IPLC x1.0`
- `日本 IEPL x0.8`

数字越低越划算。

## 🛠️ 购买与支付篇

### Q8：月付还是年付？

✅ 新手强烈建议月付：

- 机场行业波动大
- 防跑路风险
- 先试用再长期订阅

年付适合：
- 老牌机场
- 明显折扣（例如 TNT 年付限量包）

---

### Q9：机场会不会跑路？怎么降低风险？

建议“三条铁律”：

1. 不一次性充值太多
2. 主力 + 备用双机场
3. 优先选运营多年、有口碑的机场

## 🔧 故障排查篇（救命指南）

### Q10：昨天能用，今天全红超时？

按顺序排查：

1. 流量是否用完 / 套餐是否过期
2. 客户端点击“更新订阅”
3. 系统时间必须准确（误差 < 90 秒）
4. Clash 重启内核或切换节点

---

### Q11：国内网站打不开（淘宝、微信异常）？

你开了全局模式。

✅ 切回：

- Rule 模式 / PAC 模式

规则模式原理：

- 国外走代理
- 国内直连

---

### Q12：速度慢怎么办？

1. 换节点（不要只盯香港）
2. 晚高峰尝试日本/新加坡
3. 检查本地宽带是否本身就慢
4. 不要只看延迟（ms 低 ≠ 速度快）

---

### Q13：订阅链接安全吗？能随便转换吗？

⚠️ 订阅链接 = 账号密码

不要随便用在线“订阅转换网站”，风险极高：

- 被盗链
- 被刷流量
- 封号

✅ 最安全：
- 用机场原生 Clash / Shadowrocket 订阅

---

### Q14：DNS 泄露是什么？要开加密 DNS 吗？

DNS 泄露会暴露你访问了哪些网站。

✅ 建议开启：

- Clash 的 `Fake-IP`
- 或使用 `DoH/DoT`

新手无需深究，只要用推荐客户端默认配置即可。

## 🛡️ 安全与隐私篇

### Q15：机场会记录我的隐私吗？

- 机场只能看到你传输数据量
- HTTPS 加密下看不到内容

⚠️ 极敏感操作（银行/政府）建议关闭代理直连。
