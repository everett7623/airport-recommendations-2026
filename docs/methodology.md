# 📐 机场推荐评分与筛选标准 (Methodology)

> **说明：** 本文档详细说明本项目收录、评分和筛选机场的标准与流程，确保推荐透明、可复现。

---

## 📊 评分维度

每个机场从以下 5 个维度综合评估，满分 5 星：

| 维度 | 权重 | 说明 |
|------|------|------|
| **稳定性** | ⭐⭐⭐⭐⭐ | 晚高峰（20:00–23:00）节点可用率、丢包率、断线频率 |
| **速度** | ⭐⭐⭐⭐ | 带宽吞吐、延迟（Latency）、4K 视频缓冲表现 |
| **流媒体解锁** | ⭐⭐⭐⭐ | Netflix、Disney+、TikTok、ChatGPT、YouTube Premium 解锁能力 |
| **性价比** | ⭐⭐⭐ | 单位流量价格、套餐灵活性、优惠力度 |
| **服务与信誉** | ⭐⭐⭐ | 运营时长、客服响应、社区口碑、工单处理速度 |

## 🔬 测试方法

### 测试环境
- **网络：** 中国电信 / 中国联通 / 中国移动 三网分别测试
- **时间：** 晚高峰（20:00–23:00 CST）与 非高峰（09:00–11:00 CST）各一轮
- **客户端：** Clash Verge Rev / Shadowrocket 标准配置
- **周期：** 每月至少一轮全面测试

### 测试指标
1. **延迟 (Latency)：** ICMP Ping + TCPing，取 P95 值
2. **带宽 (Bandwidth)：** Speedtest.net / fast.com，取 3 次平均值
3. **丢包率 (Packet Loss)：** 持续 5 分钟 ping，记录丢包百分比
4. **流媒体解锁：** 逐平台实测（Netflix / Disney+ / TikTok / ChatGPT / BBC iPlayer）
5. **节点可用率：** 24 小时内每小时检测一次，计算在线率

## ✅ 收录标准

机场需满足以下**全部条件**方可收录：

1. **可访问性：** 官网可正常打开，订阅链接有效
2. **可用性：** 至少 3 个不同地区节点稳定在线
3. **支付便利：** 支持支付宝 / 微信支付
4. **透明度：** 明确标注线路类型、倍率、流量限制
5. **合规性：** 有基本的服务条款与隐私说明

---

## ⛔ 下架标准

触发以下**任一条件**即从推荐列表中移除（详见 [blacklist.md](blacklist.md)）：

1. 官网 / 节点持续不可用 ≥ 30 天
2. 用户大量反馈异常扣费、封号
3. 运营团队失联，公告长期不更新
4. 频繁更换域名且无合理解释
5. 社区普遍认为风险过高
6. 出现严重隐私或安全争议

## 🔄 更新流程

1. **每周自动同步：** GitHub Actions 每周从 VPSKnow Astro 源同步机场清单、价格、优惠码、分类和下架状态
2. **月度人工复核：** 每月 1–7 日对重点收录机场进行一轮测试，复核节点状态和套餐信息
3. **即时更新：** 收到可靠的下架/跑路/恢复报告后 48 小时内验证，并在 VPSKnow 数据源更新后同步到本仓
4. **季度深度评测：** 每季度对新机场进行完整的 5 维度测试
5. **社区反馈：** GitHub Issues 和 Telegram 群组的有效反馈会在下一轮更新中验证
